About Authify

A simple and secure user authentication app using Flask & MySQL.

  • Sign up with form validation
  • Login with email and password
  • Session-based authentication
  • Protected dashboard route
  • Flash messages for user feedback
  • Dynamic navbar (shows login/signup or logout)
  • Bootstrap 5 styling for responsive UI

  • Flask — Python web framework
  • MySQL — Relational database
  • Bootstrap 5 — Frontend framework
  • Flask-MySQLdb — MySQL integration for Flask

  1. Clone the repository
  2. Set up the MySQL database flask_auth
  3. Install dependencies: pip install -r requirements.txt
  4. Run the server: python run.py
  5. Visit http://localhost:5000 in your browser
